## 🧠 What I Learned

<details>
<summary><b>⚡ Swift Programming</b></summary>
<br/>

- Variables & constants, data types, control flow
- Functions, optionals, structs & classes
- Protocols, extensions, closures
- Error handling & Result type
- Functional programming — `map`, `filter`, `compactMap`

</details>

<details>
<summary><b>🎨 SwiftUI Development</b></summary>
<br/>

- Views, modifiers, property wrappers
- Layout stacks — `VStack`, `HStack`, `ZStack`
- `NavigationStack`, Lists, Forms
- Animations, gestures, `GeometryReader`
- Custom views & advanced layouts

</details>

<details>
<summary><b>📱 iOS Frameworks</b></summary>
<br/>

- **MapKit** — interactive maps & annotations
- **Core Image** — photo filters & processing
- **URLSession** — networking & REST APIs
- **Codable** — JSON encoding & decoding
- **UserNotifications** — local push notifications
- **SwiftData** — modern data persistence
- **QR Code scanning**, camera, haptics & accessibility

</details>
