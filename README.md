# 🚀 100 Days of SwiftUI – My iOS Development Journey

Hi 👋
I'm **Shivansh Omer**, an aspiring **iOS Developer from India 🇮🇳**.

This repository contains all the projects I built while completing the **100 Days of SwiftUI** course by **Paul Hudson**.

During this challenge, I built real iOS applications, practiced Swift daily, and learned modern **SwiftUI development**.

---

# 📚 About the Challenge

The **100 Days of SwiftUI** challenge focuses on learning iOS development by building real apps.

It covers:

* Swift programming fundamentals
* SwiftUI interface development
* Real iOS applications
* Data storage
* Networking
* Apple frameworks
* Advanced SwiftUI techniques

Over **100 days**, I completed:

* ✅ **19 major projects**
* ✅ Many mini challenges
* ✅ A final exam with **100 questions**
